# Gallery Android  

Simple photo viewer with koin mobile architecture of DNC company.  
Tools and etc.: MVVM+Clean, Coroutines, LiveData, Room, Koin, Retrofit+OkHttp, Glide, Jetpack navigation, Firebase.
